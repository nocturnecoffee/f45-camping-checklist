# BMW F45 兩人一貓車宿露營設備清單

這是一個單頁 GitHub Pages 靜態網站，用來分享 BMW F45 兩人一貓車宿露營設備清單。

## 建議 repository 名稱

`f45-camping-checklist`

部署後網址會是：

```text
https://nocturnecoffee.github.io/f45-camping-checklist/
```

## 檔案

- `index.html`：網站首頁
- `.nojekyll`：避免 GitHub Pages 使用 Jekyll 處理檔案
- `README.md`：說明文件
- `DEPLOY_WITH_GH_CLI.md`：GitHub CLI 部署指令

## 用 GitHub 網頁介面部署

1. 建立新的 public repository，例如 `f45-camping-checklist`。
2. 上傳本資料夾內所有檔案到 repository 根目錄。
3. 進入 repo 的 **Settings → Pages**。
4. Source 選 **Deploy from a branch**。
5. Branch 選 **main**，folder 選 **/(root)**。
6. 儲存後等待 GitHub Pages 部署完成。

## 用 GitHub CLI 部署

詳見 `DEPLOY_WITH_GH_CLI.md`。
