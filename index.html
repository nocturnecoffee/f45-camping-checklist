A. 車內睡眠系統

小計 NT$ 9,560
這個系統做什麼：把 F45 後座放倒後形成的 170×100cm 空間變成你跟女友的「車內臥室」。走訪店家後定案：放棄客製海綿床墊與海外登山短墊，改用你實際試躺過的 180×100cm 車宿專用充氣床——充飽後支撐度與舒適度都夠、洩氣後能收很小塞進車內，一次解決「客製墊好睡但收納大」「登山墊收納小但睡感差」的兩難。核心配套不變：PAMABE 延長墊補平前座椅背落差、遮陽簾擋光擋蚊。

主角：車宿專用充氣床墊 180×100cm（你今天試躺的那款）

充飽後 180×100cm、洩氣後可收成小包。選它的理由：你親自試躺過，支撐度與舒適度都過關，不像一般雙人充氣床軟塌；又能收很小，塞得進 F45。記得回填品牌型號，並確認打氣方式（內建電動幫浦／附手動幫浦）。
$5,380

1充氣床下單前的三項確認

床身 180cm 比 F45 實測睡眠面 170cm 長約 10cm，靠 PAMABE 延長墊把前座椅背端墊平就能順順攤平；寬 100cm 剛好貼齊車內寬度，下單前確認不會頂到兩側門板。第三項是打氣：山上用電動幫浦要能接 ADAM 動力線、車充或行動電源，不能只靠家用插頭。
出發前先在車內充飽一次實測必做

2延長墊（補平前座與後座的高度差）

F45 後座放倒後，跟前座椅背中間會有 20-30cm 的高度落差。把延長墊用扣具掛在兩個前座椅背後，與後座放倒形成同一平面，充氣床鋪上去床尾才不會懸空下陷。充氣床雖有支撐度，落差還是要靠延長墊撐住。

睡覺時隱私 + 防蚊蟲 + 早上不被陽光照醒。F45 太冷門，通用磁吸款貼合度差，要選車型專用 3D 開版款。下單前務必私訊賣家確認 2015 218d 對應。
蝦皮 BMW 2 系 F45 磁吸遮陽簾（你找到的賣場）~$2,000

4厚床單 + 充氣床收納袋

充氣床面直接睡會悶、貓爪也可能刺破，鋪一層厚床單保護床面兼吸濕。充氣床多半附自己的收納袋，若沒有再買黑色防水收納袋。床單用家裡的就好。
床單用家裡的、收納袋依充氣床附件決定~$600

⚠ 顧問建議與陷阱

  • 出發前一定先在車內充飽一次：確認 180cm 床身在後座放倒 + 延長墊後攤得平、前座端不懸空、兩側不頂門板。不要到營地才第一次充。
  • 充氣床山上會偷體溫：床墊內部是空氣，1000m 夜晚冷空氣會從床下對流帶走體溫。床下墊一層巧拼或鋁箔隔熱墊，或床上加厚墊被，比多蓋一條棉被更有效。
  • 怕刺破，帶補胎片：充氣床最大風險是漏氣。隨車放原廠補胎片，貓上車前先剪指甲，床面鋪厚床單當保護層。
  • 打氣電力要先想好：電動幫浦在山上要能用 ADAM 動力線或行動電源供電；手動幫浦則先確認體力與時間。半夜稍微洩氣要能快速補充。
  • 遮陽簾陷阱：F45 後三角窗形狀特殊，下單前要對方確認 7 片包含後三角窗 + 兩側後車門 + 兩側前車門 + 後尾門 + 前擋（共 7 片）。少於 7 片不要買。

B. YAKIMA 車邊帳系統（最重要、也最大筆的系統）

小計 NT$ 55,840
這個系統做什麼：車邊帳是你下車後的客廳 + 廚房 + 貓房 + 雨備區。走訪店家後大改：放棄 OWL VanShelter TC 落地帳，升級為 YAKIMA 車頂橫桿 + OverOut 270XL 扇形車邊帳 + XL 扇形帷幕的整套系統。270° 扇形帳從車側一路掃到車尾，圍出的空間非常大（你已在店裡確認）。差別是：YAKIMA 收在車頂硬殼裡，到營地「拉出來」就成形、收得快、結構硬挺；代價是貴一倍多、且橫桿要永久鎖在車頂（原本「車頂永遠空著」的原則因此改變）。

主角：YAKIMA OverOut 270XL 扇形車邊帳 + XL 扇形帷幕（含車頂橫桿）

⚠ 顧問建議與陷阱

  • 橫桿是永久裝備了：裝上 YAKIMA 橫桿後車頂不再淨空。車總高度增加約 10-15cm，進地下停車場、洗車機、平交道前先看限高；油耗也會略增（高速風阻）。長期不用可拆，但每次裝拆費工，建議常駐。
  • 車頂載重：F45 動態車頂載重約 75kg。橫桿 + 270XL 扇形帳硬殼自重約 25-40kg（XL 比一般款重），仍在範圍內；但行駛中不要再往車頂加重物，高速橫風的感受會比以前明顯。
  • 停車方向要先想好：270° 扇形帳會從車側一路掃到車尾。停車時不只選開哪一側，車尾後方也要留出扇形掃過的淨空——背風、面景觀、後方無樹無柱的方位。停錯方向整套要重停。
  • 強風要收：扇形帳面積大、吃風也大，撐開後務必確實下釘 + 收緊 guy line。風大（6 級以上）寧可收起來，鋁臂被吹凹的維修費比一頂帳還貴。
  • 空間夠大，要管的是秩序：270XL + XL 扇形帷幕圍起的空間非常大（你已確認），兩張 Windswaves 椅 + Camping Ace 五單位桌 + 匠俱貓籠 + 料理區都放得下、不必把貓籠移出去。空間大的代價是扇形外緣的下釘點與 guy line 變多，每個支點都要確實固定，不然大面積帳布吃風會晃。
  • 下釘角度：鍛造釘朝外傾斜 45 度（不是垂直），才能用槓桿原理對抗繩子拉力。垂直插的釘子拉一下就鬆。

⚠ 顧問建議與陷阱

  • 貓籠位置：放在外帳裡，但要遠離爐火區（廚房側）至少 1.5m，避免油煙嗆貓 + 火源風險。
  • 白天不要把貓單獨留車內：F45 黑色車身夏天日曬 30 分鐘可達 50°C，密閉車內 5 分鐘就可能讓貓中暑。出去用餐/上廁所要把貓放外帳貓籠+ SwitchBot 監控。
  • 夜間貓睡哪（依氣溫決定）：
    • ≥ 18°C（夏季 & 春秋）：貓睡外帳貓籠 + 自發熱毛毯
    • 12-18°C（春秋夜）：外帳貓籠 + LovePet 電熱毯通電
    • ≤ 12°C（冬夜）：直接把貓接到車內外出籠（已有），跟人睡更安全，加蓋棉被
  • 電熱毯禁忌：不要直接接觸貓皮膚（會低溫燙傷）。一定要在電熱毯上加一層毛毯/小毯子，貓睡毛毯上不是電熱毯上。連續使用不超過 8 小時，每 2 小時人工檢查貓的狀態。
  • 適應訓練：第一次帶貓出去前，至少先在家放貓籠 1 週讓貓熟悉。直接帶出去貓會壓力大叫整晚。
  • 不要省 HOOMENT 急救包：你不能省這個。買一個放車內隨車。

    • YAKIMA 在家附近空地拉出 / 收回一次，熟悉流程
    • 貓籠展開、束帶與旋鎖扣都先試綁一次

裝車 + 搭設流程

實戰流程：從家裡出發到營地紮營完成，總共約 50-75 分鐘。YAKIMA 車邊帳收在車頂硬殼裡，到營地「拉出來」比落地帳快不少。提前一晚先把車內物品裝好。

🚙 裝車邏輯（重物低 + 隨手物近）

  1. 車頂：YAKIMA 橫桿 + 車邊帳硬殼（常駐，不需每次裝拆）。出發前確認硬殼扣具鎖緊。
  2. 後廂底部：匠俱貓籠（摺疊薄板）平放鋪底；上層放 THOR 75L 收納箱（靠椅背側、重物在箱底）。
  3. 後座放倒區：車宿充氣床（洩氣收納包）+ PAMABE 延長墊 + 棉被 30L 防水袋 + Camping Ace 桌 + 兩張 Windswaves 椅收納袋。
  4. 副駕腳邊：多功能鉗 + 急救包 + 行車前隨手物（手機架、水壺）。
  5. 副駕座位：貓行車外出籠（用安全帶固定）。
  6. 注意總高：裝了 YAKIMA 橫桿後車變高，行經地下停車場、洗車機、平交道前先看限高。

⛺ 搭設順序（25-40 分鐘）

  1. 停車。先決定 270 扇形帳掃向——選一側 + 車尾後方都要留出扇形淨空，朝背風、面景觀的方位。
  2. 從車頂硬殼拉出 YAKIMA 車邊帳，展開支臂、伸出支腳，調整到水平。
  3. 下遮陽帳支腳的 30cm 鍛造釘 + 拉 guy line 收緊（強風時這步不能省）。
  4. 掛上帷幕圍出封閉空間，帷幕下緣下釘固定。
  5. 鋪地布（整片）→ 在人活動區疊巧拼／瑜伽墊。
  6. 展開匠俱貓籠，用束帶綁到 YAKIMA 支腳或車身固定點，門扣加 D 型旋鎖扣。
  7. 架 Camping Ace 五單位桌 + OWL 瀝水袋 + 兩張 Windswaves 椅。
  8. 沿橫桿／帳骨黏貼或吊掛 LED 燈條、夾 65A+ 在貓籠旁與椅子旁。
  9. 架 ADAM 動力線從營地市電拉進帳內，接燈條／風扇／充氣床幫浦。
  10. 車內：放倒後座 + 副駕 → 鋪 PAMABE 延長墊扣到前座椅背 → 充飽車宿充氣床，確認前座端不懸空、兩側不頂門板。
  11. 掛 BMW F45 磁吸遮陽簾（睡覺前才掛，白天車內透氣）。
  12. 把貓從行車外出籠放進貓籠，加 D 型旋鎖扣 + 給水給乾糧。
  13. SwitchBot 一顆放貓籠內、一顆放車內。打開 App 確認連線。

⚠ 收營時順序反過來

  • 先把貓接回行車外出籠 → 收貓籠（摺疊）→ 收椅子桌子燈具 → 小物先回 THOR 75L → 收帷幕 → 收回 YAKIMA 車邊帳入硬殼（先收 guy line 拔釘，再收支臂）→ 收地布 → 充氣床洩氣收折 → 車內物品歸位 → 最後檢查鍛造釘有沒有撿光（最常忘記）。
  • YAKIMA 收回前確認帳布乾燥：濕著收進硬殼會發霉。下雨收營若帳布濕，回家一定要再拉出來陰乾。
